# XmonadConfig
My configs for Xmonad (wallpaper included)

Just drop these into your home folder and then reboot to apply.

## Requirements:
* Xmonad
* Stalonetray
* picom (Jonaburg's fork)
* conky
* neofetch
* fish (the Friendly Interactive Shell)
